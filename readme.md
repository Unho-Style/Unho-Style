# 운호스(Unho-Style)
운호고 학생간 중고거래를 위한 중고거래 플랫폼

## 프로젝트 이름의 의미
운호스는 운호고등학교의 운호와 영어 스타일을 합쳐서 만들었어요.<br>
운호고등학교의 학생들이 자신의 스타일을 공유하고, 즐기고, 향상시키기 위해 필요한 물건들을 쉽게 구매하고 판매할 수 있는 중고마켓이 되겠다는 의미에요.<br>
다양한 카테고리의 물건들을 사고 팔며 친구들과 소통하고 당신의 새로운 모습을 발견해보세요!<br>
